# House Price Prediction using Linear Regression

## Objective
Predict house prices using:
- Square Footage
- Number of Bedrooms
- Number of Bathrooms

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Google Colab

## Dataset
House Prices - Advanced Regression Techniques (Kaggle)

## Model
Linear Regression

## Evaluation Metrics
- MAE
- MSE
- R² Score
